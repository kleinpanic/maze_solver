# Security Policy

## Supported Versions

The `main` branch is the supported line for fixes.

## Reporting

Please report vulnerabilities through GitHub private vulnerability reporting when available, or open a concise issue that avoids publishing exploit details. Include affected versions, reproduction steps, and the expected impact.

## Project Scope

Maze Solver is a local educational application and static WebUI. It does not require secrets at runtime, does not collect telemetry, and does not store user data.
